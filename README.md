# SISY - Visibility Infrastructure for Female Expertise

SISY helps women become visible as experts, speakers, and leaders across all industries. It enables organizations to discover diverse voices beyond existing networks.

## What is SISY?

SISY is two complementary things:

- **SISY Platform** - A product/SaaS platform for female speaker and expert visibility. Expert profiles, opportunity matching, visibility support, community, and organizer tools.
- **Sisy Framework** - An open-source automation framework (SisyCore, SisyCLI, SisyAPI, SisyBench, SisyBot) that packages repeatable patterns into a portable, reusable substrate.

## Website

This repo contains the landing site for [sisy.social](https://sisy.social), built with [Hugo](https://gohugo.io/).

### Pages

- `/` - Home (overview with teasers)
- `/platform/` - Full SISY Platform details
- `/framework/` - Sisy Framework documentation
- `/about/` - Story, founders, vision, and mission

### Tech Stack

- Hugo 0.142.0 (static site generator)
- Custom CSS (no frameworks)
- Lucide icons (via CDN)
- Google Fonts (Inter + Playfair Display)
- Deployed to GitHub Pages

### Local Development

```bash
hugo server -D
```

### Build

```bash
hugo --gc --minify
```
